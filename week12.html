Week 12: JS Arrays & Dynamic Pages

This week introduces **arrays and dynamic page updates**, allowing multiple values to be stored, iterated, and displayed interactively.

Day 1: Creating Arrays

Learn: Arrays store multiple values in a single variable.

let fruits = ["apple", "banana", "orange"];
console.log(fruits);

Do: Create an array with your top 5 favorite foods.

Tip: Arrays are zero-indexed: the first item is at index 0.
Day 2: Accessing & Modifying Arrays

Learn: Access items by index and modify values.

console.log(fruits[1]); // banana
fruits[0] = "mango";

Do: Change the first item in your array and log it.

Tip: Use fruits.length to find array size and avoid going out-of-bounds.
Day 3: Array Methods (push, pop, shift, unshift)

Learn: Easily add/remove elements:

fruits.push("grape");    // add end
fruits.pop();             // remove end
fruits.unshift("kiwi");   // add start
fruits.shift();           // remove start

Do: Manipulate your array and log changes each time.

Tip: push/pop affect the end; unshift/shift affect the start.
Day 4: Looping Through Arrays

Learn: Use loops to process array items.

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

Do: Log all items with a message like "I like ___".

Tip: You can also use forEach for cleaner iteration.
Day 5: Display Array Items on Page

Learn: Use the DOM to display array content dynamically.

let list = document.getElementById("myList");
fruits.forEach(function(fruit) {
  list.innerHTML += "<li>" + fruit + "</li>";
});

Do: Create <ul id="myList"></ul> and dynamically add array items.

Tip: Using innerHTML += works for small lists; for large lists, consider creating elements with document.createElement.
Day 6: Interactive Array Updates

Learn: Add items to arrays via input fields and buttons.

<input id="fruitInput">
<button onclick="addFruit()">Add</button>

<script>
let fruits = [];
function addFruit() {
  let val = document.getElementById("fruitInput").value;
  fruits.push(val);
  console.log(fruits);
}
</script>

Do: Create input and button to add items to your array dynamically.

Tip: Always check that the input is not empty before adding.
Day 7: Review & Mini Dynamic Page

Learn: Combine arrays, loops, and DOM updates for interactive pages.

Do: Build a simple dynamic shopping list: enter items, click "Add", and see them appear on the page.

Tip: Clear the input field after adding to improve user experience.

Check: Browser updates correctly. This completes Week 12.
