This week teaches **events, form inputs, and basic interaction**, helping you build dynamic pages that respond to user actions.

Day 1: Click Events Review

Learn: Buttons and elements can react to clicks using onclick or addEventListener. See:

<button id="btn">Click Me</button>
<script>
document.getElementById("btn").addEventListener("click", function() {
  alert("Button clicked!");
});
</script>
Do: Create a button that shows your name in an alert on click. Check: Browser shows alert when button is clicked.

Day 2: Input Fields

Learn: HTML forms allow users to enter data, which JS can access. See:

<input id="nameInput" placeholder="Enter name">
<button onclick="showName()">Submit</button>
<script>
function showName() {
  let name = document.getElementById("nameInput").value;
  alert("Hello " + name);
}
</script>
Do: Add an input for favorite hobby and display it in an alert. Check: Browser shows correct message.

Day 3: Change & Input Events

Learn: Input fields can react as the user types using oninput or onchange. See:

<input id="textInput">
<p id="display"></p>
<script>
document.getElementById("textInput").addEventListener("input", function() {
  document.getElementById("display").innerText = this.value;
});
</script>
Do: Display typed text live in a paragraph. Check: Browser updates as you type.

Day 4: Form Validation

Learn: JS can validate input before submitting. See:

function checkForm() {
  let age = document.getElementById("age").value;
  if(age === "" || age < 0) {
    alert("Enter a valid age");
    return false;
  }
  return true;
}
<form onsubmit="return checkForm()">
<input id="age" type="number">
<button>Submit</button>
</form>
Do: Create a form asking for username and prevent submission if empty. Check: Browser blocks empty input and shows alert.

Day 5: Mouse Events

Learn: Elements can respond to mouseover, mouseout, mousedown, mouseup. See:

let box = document.getElementById("box");
box.addEventListener("mouseover", () => box.style.background = "lightblue");
box.addEventListener("mouseout", () => box.style.background = "white");
Do: Create a div that changes color on hover. Check: Browser shows color changes correctly.

Day 6: Keyboard Events

Learn: Detect key presses with keydown or keyup. See:

document.addEventListener("keydown", function(e) {
  console.log("Key pressed: " + e.key);
});
Do: Log letters pressed in console. Check: Browser console shows keys pressed.

Day 7: Review & Interactive Form

Learn: Combine input, events, and validation. Do: Build a mini sign-up form asking name and age, display welcome message only if age is valid. Check: Browser prevents invalid input and displays correct messages. This completes Week 13.
