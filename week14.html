This week extends DOM events and forms into **interactive, dynamic page features**, including lists, adding/removing items, and live updates.

Day 1: Dynamic List Addition

Learn: Add items to a list dynamically using inputs and buttons. See:

<input id="itemInput">
<button onclick="addItem()">Add</button>
<ul id="list"></ul>
<script>
function addItem() {
  let val = document.getElementById("itemInput").value;
  if(val) document.getElementById("list").innerHTML += "<li>" + val + "</li>";
}
</script>
Do: Add a task to your list using input field. Check: Browser shows item in list.

Day 2: Removing List Items

Learn: Remove items from the DOM using JS. See:

function removeLast() {
  let list = document.getElementById("list");
  list.removeChild(list.lastElementChild);
}
<button onclick="removeLast()">Remove Last</button>
Do: Add and remove items dynamically. Check: Browser updates list correctly.

Day 3: Toggling Classes

Learn: Toggle CSS classes on events for styling changes. See:

<div id="box">Click me</div>
<script>
document.getElementById("box").addEventListener("click", function() {
  this.classList.toggle("active");
});
</script>
Do: Create a div that changes style when clicked. Check: Browser toggles CSS class correctly.

Day 4: Form Input Live Preview

Learn: Show live updates from inputs to page elements. See:

<input id="name"><p id="preview"></p>
<script>
document.getElementById("name").addEventListener("input", function() {
  document.getElementById("preview").innerText = this.value;
});
</script>
Do: Type your username and see it appear live on the page. Check: Browser shows live preview.

Day 5: Multiple Event Listeners

Learn: Attach multiple event listeners to one element. See:

let box = document.getElementById("box");
box.addEventListener("mouseover", () => box.style.background="yellow");
box.addEventListener("mouseout", () => box.style.background="white");
Do: Make a box change color on hover and revert when mouse leaves. Check: Browser shows behavior.

Day 6: Form Submission with JS

Learn: Prevent default submission to handle form via JS. See:

<form id="form">
  <input id="email">
  <button>Submit</button>
</form>
<script>
document.getElementById("form").addEventListener("submit", function(e) {
  e.preventDefault();
  alert("Submitted: " + document.getElementById("email").value);
});
</script>
Do: Create a simple email form and alert value on submit. Check: Browser prevents refresh and shows value.

Day 7: Review & Interactive Page

Learn: Combine DOM events, form inputs, and live updates. Do: Build a small interactive todo list or signup form with add/remove functionality, live previews, and alerts for invalid inputs. Check: Browser demonstrates all features correctly. This completes Week 14.
