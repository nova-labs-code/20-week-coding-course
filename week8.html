Week 8: CSS Grid & Responsive

This week teaches CSS Grid for powerful layouts and responsive techniques to adapt pages to different screens.

Day 1: Introduction to CSS Grid

Learn: Grid allows 2D layouts. Example:

.container {display: grid; grid-template-columns: 1fr 1fr 1fr; gap:10px;}
Do: Create a container with 3 columns and 3 boxes. Check: Open the browser to see boxes aligned in a grid.

Day 2: Grid Rows & Columns

Learn: You can define explicit row and column sizes using grid-template-rows and grid-template-columns. Example:

.container {grid-template-columns: 100px 200px 100px; grid-template-rows: 50px 100px;}
Do: Create a container with three columns of different widths and two rows of different heights. Check: Open the browser and confirm boxes match the specified row heights and column widths.

Day 3: Grid Gaps & Alignment

Learn: Use gap for spacing between rows and columns. Align content using justify-items and align-items. Example:

.container {display: grid; grid-template-columns: repeat(3, 1fr); gap: 15px; justify-items: center; align-items: center;}
Do: Apply gaps between grid items and align them in the center. Check: Open in a browser to see proper spacing and alignment of items.

Day 4: Grid Areas

Learn: Named grid areas allow you to assign boxes to specific regions of the grid. Example:

.container {
  display: grid;
  grid-template-areas: 
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-gap:10px;
}
.header {grid-area: header;}
.sidebar {grid-area: sidebar;}
.main {grid-area: main;}
.footer {grid-area: footer;}
Do: Create a layout using grid areas for header, sidebar, main, and footer. Check: Open browser to verify items occupy the correct grid regions.

Day 5: Responsive Grid

Learn: Use media queries to make grid layouts adapt to screen size. Example:

@media (max-width: 600px) {
  .container {grid-template-columns: 1fr; grid-template-rows: auto;}
}
Do: Make your 3-column layout stack into 1 column on small screens. Check: Resize the browser or use device mode to ensure the layout responds correctly.

Day 6: Grid Templates & Repeat

Learn: The repeat() function simplifies grid templates. Example:

.container {grid-template-columns: repeat(3, 1fr); grid-template-rows: repeat(2, 150px);}
Do: Apply repeat to your columns and rows for concise code. Check: Open in the browser to see equal columns and rows set by repeat.

Day 7: Review & Responsive Grid Page

Learn: Combine everything learned in Week 8: explicit rows/columns, gaps, alignment, named grid areas, and responsive media queries. Do: Build a responsive profile page with header, main content, sidebar, and footer using CSS Grid. Check: Open the browser to verify items are correctly aligned, spaced, and adapt to different screen sizes. This completes Week 8.
