Week 8: CSS Grid & Responsive

This week focuses on **CSS Grid layouts** and responsive techniques to build flexible, modern web pages that adapt to any screen size. Lessons include examples, exercises, and tips to solidify understanding of rows, columns, gaps, alignment, and media queries.

Day 1: Introduction to CSS Grid

Learn: CSS Grid allows you to build 2D layouts with rows and columns. Itâ€™s more flexible than float or Flexbox for complex layouts. Use display: grid; to start a container.

See:

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}

Do: Create a container with 3 columns and 3 boxes. Try resizing the browser to see how items stay aligned in the grid.

Check: Open browser to see boxes aligned in a uniform 3-column layout.

Tip: Use 1fr to make columns equally divide available space.
Day 2: Grid Rows & Columns

Learn: Define specific sizes for rows and columns with grid-template-rows and grid-template-columns. You can mix pixels, fractions, and percentages for precise control.

See:

.container {
  grid-template-columns: 100px 200px 100px;
  grid-template-rows: 50px 100px;
}

Do: Build a grid with 3 columns of different widths and 2 rows of different heights. Experiment with mixing px and fr units.

Check: Browser displays boxes matching row heights and column widths as defined.

Tip: Use fractions (fr) for responsive proportional sizing rather than fixed pixels.
Day 3: Grid Gaps & Alignment

Learn: The gap property sets spacing between grid items. Control item alignment with justify-items (horizontal) and align-items (vertical).

See:

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 15px;
  justify-items: center;
  align-items: center;
}

Do: Add gaps between items and center them horizontally and vertically.

Check: Browser shows items spaced evenly and aligned in the center of their grid cells.

Tip: Combine justify-items and align-items for complete control of item placement.
Day 4: Grid Areas

Learn: Named grid areas let you assign items to specific regions, making layouts easier to manage than positioning individual cells.

See:

.container {
  display: grid;
  grid-template-areas: 
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-gap: 10px;
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Do: Create a layout with header, sidebar, main content, and footer using named areas.

Check: Browser shows items in the correct regions without using complicated positioning.

Tip: Named areas make it easier to change layouts later without rewriting HTML.
Day 5: Responsive Grid

Learn: Use media queries to make grids adapt to different screen sizes. Change column layout based on device width.

See:

@media (max-width: 600px) {
  .container {
    grid-template-columns: 1fr;
    grid-template-rows: auto;
  }
}

Do: Make your 3-column layout collapse into 1 column on smaller screens.

Check: Resize browser or use developer tools to see layout adjust for mobile.

Tip: Combine responsive grids with fr units for scalable layouts.
Day 6: Grid Templates & Repeat

Learn: The repeat() function simplifies code when you need multiple rows or columns of the same size.

See:

.container {
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(2, 150px);
}

Do: Use repeat() for your columns and rows instead of writing each individually.

Check: Browser shows grid with equally sized columns and rows set by repeat function.

Tip: Use repeat(auto-fit, minmax(150px, 1fr)) for fully responsive grids.
Day 7: Review & Responsive Grid Page

Learn: Combine all Week 8 techniques: explicit rows/columns, gaps, alignment, named grid areas, repeat function, and responsive media queries. Build a professional page with a header, main content, sidebar, and footer using CSS Grid.

Do:

  • Create a container with header, main, sidebar, and footer using grid areas.
  • Use gaps and alignment to space items nicely.
  • Apply repeat() and fractional units for consistent sizing.
  • Add a media query to make the layout responsive for mobile.

Check: Browser shows a clean, responsive page where all items align correctly and adapt to different screen widths. Congratulations, Week 8 is complete!

Tip: Use CSS Grid for complex layouts instead of floats; it simplifies responsive design and reduces extra CSS.
