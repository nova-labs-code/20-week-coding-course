This week introduces **JavaScript fundamentals** to add interactivity. Each day is ~30 minutes with examples to try in the browser console or in script tags.

Day 1: Introduction to JavaScript & Console

Learn: JavaScript adds dynamic behavior to web pages. Use console.log() to print messages for debugging. See:

console.log("Hello, world!");
Do: Open your browser console and type console.log("My name is [Your Name]"). Check: Verify the message appears in the console.

Day 2: Variables (let, const, var)

Learn: Variables store data. Use let for changeable values, const for constants, var (older). See:

let age = 15;
const name = "Elisha";
var hobby = "coding";
console.log(name, age, hobby);
Do: Create 3 variables for your favorite color, food, and game. Check: Log them to the console to confirm.

Day 3: Data Types & typeof

Learn: JS data types include string, number, boolean, object, undefined. Use typeof to check type. See:

let score = 100;
let isOnline = true;
console.log(typeof score);  // number
console.log(typeof isOnline); // boolean
Do: Create 2 variables of different types and log their types. Check: Verify results in the console.

Day 4: Basic Arithmetic

Learn: JS supports arithmetic: +, -, *, /, %. See:

let x = 10;
let y = 3;
console.log(x + y); // 13
console.log(x % y); // 1
Do: Calculate your age next year, or the remainder when dividing two numbers. Check: Verify results in console.

Day 5: Strings & Concatenation

Learn: Strings can be combined with + or template literals `Hello ${name}`. See:

let firstName = "Elisha";
let lastName = "Samson";
console.log("Hello " + firstName + " " + lastName);
console.log(`Hello ${firstName} ${lastName}`);
Do: Create a greeting message with your name using both methods. Check: Console shows the correct greeting.

Day 6: User Interaction with prompt

Learn: Prompt collects user input. See:

let age = prompt("Enter your age:");
console.log("You are " + age + " years old.");
Do: Prompt for your favorite hobby and display it in console. Check: Enter values to verify prompt works.

Day 7: Review & Mini JS Console Exercise

Learn: Combine variables, types, arithmetic, strings, and prompt to create small console interactions. Do: Write a program asking for name and age, then log a message like "Hello [Name], next year you will be [age+1]". Check: Open console to see if program works correctly. This completes Week 9.
