Week 11: JS Conditionals & Loops

This week focuses on **decision making and loops** in JavaScript, letting your programs respond dynamically to conditions and repeat tasks efficiently.

Day 1: If Statements

Learn: if statements execute code only when a condition is true. Use else for fallback behavior.

let age = 16;
if (age >= 18) {
  console.log("You can vote");
} else {
  console.log("Too young to vote");
}

Do: Create a variable for your age and log a message based on whether you’re a teenager or not.

Tip: Conditions must evaluate to true or false. Use comparison operators like >, <, ===, !==.
Day 2: Else If & Nested Conditions

Learn: Use else if for multiple conditions and nested if for more complex logic.

let score = 85;
if (score >= 90) {console.log("A");}
else if (score >= 80) {console.log("B");}
else {console.log("C");}

Do: Create grade messages for three ranges: A, B, C.

Tip: Nesting allows checking sub-conditions, but try to keep code readable.
Day 3: For Loops

Learn: for loops repeat code a set number of times using a counter variable.

for (let i = 1; i <= 5; i++) {
  console.log("Number " + i);
}

Do: Log numbers 1–10 using a for loop.

Tip: Loops can iterate over arrays, strings, or ranges efficiently.
Day 4: While Loops

Learn: while loops continue executing while a condition is true.

let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}

Do: Use a while loop to log even numbers between 2 and 20.

Tip: Ensure the loop variable changes; otherwise, you may create an infinite loop.
Day 5: Break & Continue

Learn: break stops a loop immediately, continue skips the current iteration.

for (let i=1; i<=10; i++) {
  if (i===5) break;
  console.log(i);
}
for (let i=1; i<=5; i++) {
  if (i===3) continue;
  console.log(i);
}

Do: Create a loop that skips multiples of 3 and stops at 12.

Tip: Use break for early exit and continue to skip unwanted iterations without exiting the loop.
Day 6: Combining Conditionals & Loops

Learn: Combine loops and conditions to handle dynamic scenarios efficiently.

for (let i=1; i<=10; i++) {
  if (i % 2 === 0) console.log(i + " is even");
  else console.log(i + " is odd");
}

Do: Log numbers 1–20 and indicate if each is even or odd.

Tip: Modulus operator % is useful for checking divisibility.
Day 7: Review & Mini Interactive Script

Learn: Combine conditionals and loops to create small programs like FizzBuzz or number classification.

Do: Ask the user for a number and print numbers from 1 to that number with:

  • "Fizz" if divisible by 3
  • "Buzz" if divisible by 5
  • "FizzBuzz" if divisible by both
  • The number itself otherwise

Check: Browser console should show correct outputs. This completes Week 11.

Tip: Try different ranges and conditions for practice. Loops plus conditionals form the backbone of logic in JS.
