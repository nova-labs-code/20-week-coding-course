Week 3: CSS Basics

Welcome to Week 3! This week we focus on **CSS styling** to make your HTML pages visually appealing. Each day follows the “Learn → See → Do → Check” method, with code examples included, and should take about 30 minutes.

Day 1: Introduction to CSS & Inline Styles

Learn: CSS (Cascading Style Sheets) adds style to HTML elements. The simplest way is using the style attribute inline. See: For example, <p style="color:blue; font-size:18px;">Blue text</p> changes text color and size. Do: Add inline styles to the paragraphs and headings of your profile page to change text color, font size, and style. Check: Open the page in a browser to see the applied styles and experiment by changing colors and sizes to understand the impact of inline CSS.

Day 2: Internal CSS & Selectors

Learn: CSS can be added internally inside a <style> tag in the <head> of your HTML. Selectors target elements like tags, classes, or IDs. See: 

<style>
  h1 {color:darkblue;}
  p {font-family: Arial; font-size:16px;}
</style>
Do: Add a <style> section in your page and style headings and paragraphs using tag selectors. Check: Make sure changes appear in the browser and experiment with different font sizes and colors.

Day 3: Classes & IDs

Learn: Classes (class) and IDs (id) let you target specific elements with CSS. See: 

<h2 class="section-title">About Me</h2>
<p id="bio">Hello!</p>

<style>
  .section-title {color:green;}
  #bio {font-style:italic;}
</style>
Do: Assign classes to your section headings and an ID to your main paragraph, then style them differently. Check: Verify in the browser that each element has the intended style.

Day 4: Colors & Backgrounds

Learn: CSS allows text and backgrounds to be styled with colors. You can use names, hex codes, or RGB. See: 

p {color: #333333; background-color: #f0f0f0;}
h1 {color: red;}
Do: Style your profile page paragraphs with a light background and headings with colors that stand out. Check: Open in a browser and experiment with different color combinations for readability.

Day 5: Fonts & Text Alignment

Learn: CSS controls font-family, size, weight, and alignment. See: 

p {font-family: 'Courier New', monospace; font-size:16px; text-align:center;}
h2 {font-weight:bold; text-align:left;}
Do: Apply different fonts to headings and paragraphs and try center, left, and right alignment. Check: Open in a browser to see how font choices and alignment affect readability and page appearance.

Day 6: Borders, Padding & Margins

Learn: CSS controls spacing and borders with border, padding, and margin. See: 

p {border: 1px solid #ccc; padding:10px; margin:15px;}
Do: Add borders around paragraphs, add padding inside, and margins outside. Check: Observe how spacing changes in the browser and experiment with different border styles and sizes.

Day 7: Review & Mini Styled Page

Learn: Combine everything from Week 3 to style your Week 1/2 profile page. Use internal CSS, classes, IDs, colors, fonts, borders, padding, and margins to make your page visually appealing and readable. Do: Build a styled profile page and Check: Open it in a browser to verify that all styles apply correctly, text is readable, sections are clear, and the layout looks organized. This final exercise consolidates all CSS basics for Week 3.
