Week 10: JS Functions, Events & DOM

This week focuses on **functions, events, and DOM manipulation** to make your pages interactive. Each day has explanations, examples, exercises, and tips for hands-on practice.

Day 1: Functions

Learn: Functions are reusable blocks of code that perform tasks. Use them to avoid repeating code.

See:

function greet(name) {
  console.log("Hello " + name);
}
greet("Elisha");

Do: Create a function to add two numbers and log the result.

Check: Console shows the correct sum.

Tip: Functions can be declared first and called later, or stored in variables as arrow functions.
Day 2: Parameters & Return Values

Learn: Functions can take inputs (parameters) and produce outputs (return values).

See:

function square(num) {
  return num * num;
}
console.log(square(5)); // 25

Do: Create a function that multiplies two numbers and returns the result.

Check: Console prints the correct value.

Tip: Use return to get a value out of a function; without it, the function returns undefined.

Learn: JavaScript can respond to user actions such as clicks, key presses, or mouse movements.

See:

<button onclick="sayHello()">Click Me</button>
<script>
function sayHello() {
  alert("Hello!");
}
</script>

Do: Add a button that shows your name in an alert when clicked.

Check: Click the button in the browser.

Tip: Inline onclick works for beginners, but using addEventListener is better practice.
Day 4: DOM Selection

Learn: JS can select HTML elements to manipulate them using getElementById, getElementsByClassName, or querySelector.

See:

let header = document.getElementById("header");
header.style.color = "red";

Do: Give a heading an ID and change its color using JS.

Check: Browser shows updated heading color.

Tip: Use descriptive IDs for easier selection.
Day 5: DOM Content Change

Learn: JS can change content dynamically with innerText or innerHTML.

See:

let msg = document.getElementById("msg");
msg.innerText = "Welcome!";

Do: Create a paragraph with an ID and a button that changes its text on click.

Check: Browser updates the text correctly.

Tip: innerHTML allows HTML formatting inside elements.
Day 6: Combining Events & DOM

Learn: You can combine click events with DOM updates to create interactive features.

See:

<button onclick="changeText()">Click</button>
<p id="demo">Old Text</p>

<script>
function changeText() {
  document.getElementById("demo").innerText = "New Text!";
}
</script>

Do: Make a button change the text of a paragraph.

Check: Verify behavior in browser.

Tip: You can update multiple elements at once inside a function for more complex interactions.
Day 7: Review & Mini Interactive Page

Learn: Combine functions, events, and DOM manipulation for full interactivity.

Do: Create a small profile card with a button that updates a message, like favorite hobby or greeting.

Check: Open browser and test button behavior. Experiment with adding colors or multiple paragraphs updating dynamically. This completes Week 10.

Tip: Practice small projects daily; DOM interactions are the foundation of web interactivity.
