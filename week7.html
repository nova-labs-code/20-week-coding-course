This week we explore **CSS transitions, animations, layering, z-index, and hover transformations** to make pages interactive and smooth. One long paragraph per day, ~30 minutes each.

Day 1: CSS Transitions

Learn: Transitions allow smooth changes of properties. Example: transition: all 0.3s ease; animates any change. See:

div.box {background: #ddd; padding:20px; transition: all 0.3s ease;}
div.box:hover {background:#aaa;}
Do: Add a hover transition to a section or card on your page. Check: Observe smooth color change when hovering.

Day 2: Transform & Scale

Learn: Use transform to rotate, scale, or move elements. Example: transform: scale(1.1); enlarges element. Do: Apply scale on hover to a card or button. Check: Open browser and hover to see interactive growth.

Day 3: Z-index & Layering

Learn: z-index controls stacking order for positioned elements. Example: position: relative; z-index: 2;. Do: Overlap two divs and assign different z-index values to see which appears on top. Check: Browser shows correct layering.

Day 4: Opacity & Visibility

Learn: CSS can control transparency with opacity or visibility. Example: opacity: 0.5; makes element semi-transparent. Do: Apply opacity to an image or div. Check: Browser shows transparency effect.

Day 5: Hover Transitions Combined

Learn: Combine transformations, color changes, and opacity with transitions for smooth hover effects. See:

div.card {
  transition: all 0.4s ease;
}
div.card:hover {
  transform: scale(1.05);
  opacity: 0.8;
  background-color: #cce7ff;
}
Do: Apply combined hover transitions to a section. Check: Observe smooth visual changes in the browser.

Day 6: Border & Shadow Animations

Learn: You can animate borders and shadows. Example: box-shadow changes with transition. Do: Add animated shadows and border color changes on hover for a card. Check: Browser shows interactive changes on hover.

Day 7: Review & Interactive Page

Learn: Combine all transitions, transformations, z-index, opacity, and shadows for a fully interactive page. Do: Enhance your profile page cards or sections with hover effects and smooth transitions. Check: Open in browser to see everything work together seamlessly. This completes Week 6.
