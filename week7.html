Week 7: CSS Typography & Media

This week focuses on **advanced typography and media integration**: fonts, line-heights, letter-spacing, text alignment, responsive images, and embedding videos. Lessons include multiple examples, exercises, and tips for best practices.

Day 1: Font Families & Sizes

Learn: CSS lets you control fonts with font-family and text size with font-size. Using a combination of web-safe fonts and custom fonts improves readability and design consistency.

See:

p { font-family: 'Courier New', monospace; font-size:16px; }
h1 { font-family: 'Georgia', serif; font-size:32px; }

Do: Apply different fonts to headings, paragraphs, and sections. Experiment with font-size for emphasis.

Check: Open in the browser and notice how font choices affect readability and style.

Tip: Use serif fonts for headings and sans-serif for body text to improve readability.
Day 2: Line-height & Letter-spacing

Learn: Adjust spacing between lines with line-height and spacing between letters with letter-spacing. Proper spacing improves readability and visual hierarchy.

See:

p { line-height: 1.8; letter-spacing: 1px; }
h2 { line-height: 1.4; letter-spacing: 2px; }

Do: Modify line-height and letter-spacing for your paragraphs and headings. Compare changes to see which combinations are easiest to read.

Check: Browser shows text with clear spacing, easier on the eyes.

Tip: Larger line-height improves readability for long paragraphs, while slight letter-spacing can improve headings.
Day 3: Text Alignment, Decoration & Transform

Learn: Control text layout with text-align, style text with text-decoration and text-transform for uppercase, lowercase, or capitalized text.

See:

h2 { text-align: center; text-transform: uppercase; }
a { text-decoration: none; color: #1e90ff; }

Do: Center headings, remove underlines from links, or capitalize certain sections.

Check: Browser shows clean, styled text with proper alignment and transformations.

Tip: Combine text-transform and font-weight to make headings visually striking.
Day 4: Responsive Images

Learn: Images should scale for different screen sizes. Use width:100%; max-width:400px; to make images responsive.

See:

img { width: 100%; max-width: 400px; height: auto; border-radius: 8px; }

Do: Add images to your page and test resizing the browser to see the images scale correctly.

Check: Browser shows images resizing without distortion, maintaining aspect ratio.

Tip: Use images that are high-quality but optimized for web to prevent slow loading.
Day 5: Embedding Videos

Learn: Add video using <video> or <iframe>. Control size and responsiveness to fit page design.

See:

<video width="320" controls>
  <source src="video.mp4" type="video/mp4">
</video>

<iframe width="560" height="315" src="https://www.youtube.com/embed/example" 
  frameborder="0" allowfullscreen></iframe>

Do: Embed a local video or YouTube video. Ensure it fits within the page layout and scales if necessary.

Check: Browser plays video correctly with controls.

Tip: Use autoplay sparingly; always include controls for user experience.
Day 6: Web-safe Fonts & Google Fonts

Learn: Use web-safe fonts for reliability or import Google Fonts for custom design. Include fonts in CSS with @import or <link> in <head>.

See:

@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

body { font-family: 'Roboto', sans-serif; }
h1 { font-family: 'Roboto', sans-serif; font-weight: 700; }

Do: Apply a Google Font to headings or body text. Compare web-safe vs Google Fonts for design impact.

Check: Browser shows custom font applied correctly.

Tip: Combine fonts wisely: one for headings, one for body text. Avoid too many fonts on the same page.
Day 7: Review & Typographic Page

Learn: Combine all Week 7 techniques: font families, sizes, line-height, letter-spacing, text alignment, responsive images, and embedded videos to create a professional page.

Do:

  • Style headings and paragraphs with fonts, size, spacing, and alignment.
  • Add at least two responsive images and one embedded video.
  • Ensure text and media scale properly across different devices.
  • Use Google Fonts for headings or body for consistent design.

Check: Open in browser; text should be readable, images scale nicely, and video functions correctly.

Tip: Consistency in typography and media placement improves UX and visual appeal.
