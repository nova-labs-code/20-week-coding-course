Week 7: CSS Typography & Media

This week focuses on typography and adding media like images and videos, using fonts, sizes, line-heights, and responsive image techniques.

Day 1: Font Families & Sizes

Learn: CSS controls fonts using font-family and font-size. See: 

p {font-family: 'Courier New', monospace; font-size:16px;}
Do: Change fonts and sizes of headings and paragraphs. Check: Open browser to see differences.

Day 2: Line-height & Letter-spacing

Learn: line-height controls spacing between lines, letter-spacing for space between letters. See: 

p {line-height:1.8; letter-spacing:1px;}
Do: Adjust spacing on paragraphs and headings. Check: Browser shows improved readability.

Day 3: Text Alignment & Decoration

Learn: CSS allows text-align, text-decoration, and text-transform. See: 

h2 {text-align:center; text-transform:uppercase;}
Do: Center headings and apply text transformations. Check: Browser shows updated typography.

Day 4: Responsive Images

Learn: Make images scale using width:100%; max-width:400px;. See: 

img {width:100%; max-width:400px;}
Do: Add images to your page and ensure they resize. Check: Resize browser to see responsiveness.

Day 5: Embedding Videos

Learn: Use <video> or <iframe> for media. See: 

<video width="320" controls><source src="video.mp4" type="video/mp4"></video>
Do: Embed a sample video or YouTube iframe. Check: Browser plays media correctly.

Day 6: Web-safe Fonts & Google Fonts

Learn: Use web-safe fonts or include Google Fonts with @import. See: 

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');
Do: Apply a Google Font to headings. Check: Browser shows new font applied.

Day 7: Review & Typographic Page

Learn: Combine all typography and media skills. Do: Enhance your profile page with styled text, responsive images, and embedded video. Check: Browser displays readable typography and properly scaled media.
