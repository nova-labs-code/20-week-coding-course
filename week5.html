Week 5: CSS Styling & Effects

This week focuses on **advanced CSS styling** to make your pages visually appealing with shadows, borders, rounded corners, gradients, typography, and hover effects. Each day is now expanded with examples, exercises, and tips to strengthen your skills.

Day 1: Rounded Corners & Borders

Learn: CSS border-radius adds rounded corners, and border controls width, style, and color.

See:

div { 
  border: 2px solid #333; 
  border-radius: 10px; 
  padding: 10px; 
  background: #fafafa;
}

Do:

  • Add a rounded border to one section of your profile page.
  • Experiment with different border-radius values like 25px, 50%, or 0.
  • Change border style to dotted, dashed, or double.

Check: Open in a browser to see how corners and borders look.

Tip: A border-radius of 50% on a square element creates a perfect circle.
Day 2: Box Shadows

Learn: Shadows give depth with box-shadow: horizontal offset, vertical offset, blur, spread, and color.

See:

div {
  box-shadow: 5px 5px 15px rgba(0,0,0,0.3);
}

Do:

  • Add a shadow to a header or card.
  • Try adjusting offsets and blur to see effects.
  • Combine with border-radius for a softer look.

Check: Open in browser and notice how elements “pop” off the page.

Tip: Shadows can simulate depth and hierarchy, improving readability.
Day 3: Text Shadows & Typography

Learn: Text shadows enhance headings; CSS controls font-family, weight, style, letter-spacing, and line-height.

See:

h1 {
  text-shadow: 2px 2px 5px gray;
  font-family: 'Verdana', sans-serif;
}
p { 
  letter-spacing: 1px; 
  line-height: 1.5; 
}

Do:

  • Add subtle shadows to headings.
  • Experiment with letter-spacing and line-height in paragraphs.
  • Try different fonts for headings and body text.

Check: Open in browser to see how typography affects readability and emphasis.

Tip: Use text-shadow lightly to improve legibility without making text hard to read.
Day 4: Gradients

Learn: Gradients create smooth color transitions. Use linear-gradient or radial-gradient.

See:

div {
  background: linear-gradient(to right, #ff7e5f, #feb47b);
}

Do:

  • Apply a gradient background to a section or card.
  • Experiment with top, bottom, left, right directions.
  • Try multiple colors in one gradient.

Check: Observe color transitions in browser.

Tip: Gradients can replace images for lighter, faster-loading backgrounds.
Day 5: Hover Effects

Learn: Hover pseudo-class :hover adds interactivity.

See:

a:hover { color: red; text-decoration: underline; }
div:hover { background-color: #eee; transform: scale(1.05); }

Do:

  • Add hover effects to links, buttons, and cards.
  • Try changing background, text color, and scale on hover.
  • Combine hover with transitions for smooth effects.

Check: Hover in browser to see interactions.

Tip: Use transition: 0.3s to animate hover smoothly.
Day 6: Combining Effects

Learn: Combine rounded corners, shadows, gradients, text shadows, and hover effects for stylish elements.

See:

div.card {
  border-radius: 10px;
  box-shadow: 2px 2px 10px rgba(0,0,0,0.2);
  background: linear-gradient(to right, #a1c4fd, #c2e9fb);
  transition: transform 0.3s;
}
div.card:hover { transform: scale(1.05); }

Do:

  • Create a card with combined effects.
  • Experiment with gradient directions, shadow intensity, and hover scaling.
  • Use consistent colors and shadows for a professional look.

Check: Open in browser to see the card “pop” visually.

Tip: Combining subtle effects improves design without overwhelming the user.
Day 7: Review & Styled Mini Page

Learn: Merge all Week 5 styling techniques into a polished profile page: rounded corners, shadows, gradients, text effects, and hover interactions.

Do:

  • Apply styles to at least three sections or cards.
  • Test hover effects for interactivity.
  • Ensure visual consistency across sections.
  • Combine typography, colors, and spacing for readability.

Check: Open in browser and verify all effects display correctly, hover works, shadows and gradients enhance the design, and the layout is visually appealing.

Tip: Use subtle, consistent styling for a modern, professional look. Avoid over-styling.
